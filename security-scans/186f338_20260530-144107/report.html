Security Scan Report - E:\wps-yundoc-admin-console

未发现高危/严重的生产前端漏洞。主要问题是开发环境暴露。

F-001 Low/Medium - Dev server exposes local backend proxy on the LAN

npm run dev 使用 vite --host 0.0.0.0，同时 /api 代理到 http://localhost:8080。同一局域网内的人可能通过开发机的 5173 端口访问本地后端代理。

建议默认改为 vite --host 127.0.0.1，需要局域网调试时再单独提供显式脚本。

  • No admin JWT/WPS token stored in session storage.
  • CSRF header is sent from the CSRF cookie.
  • Dynamic API path segments are encoded.
